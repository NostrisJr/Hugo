# Corpus — confusion « dans » / « d'en »

> Phrases originales. Phase 9 — [`crate::rules::confusion::dans_den`].
>
> Mémo : « dans » = préposition de lieu ou de temps ; « d'en » = de + en,
> toujours suivi d'un infinitif.

## dans → d'en (préposition suivie d'un infinitif)

- FAUX: il essaie dans sortir.           → d'en
- FAUX: il se garde dans parler.         → d'en
- FAUX: je tente dans savoir plus.       → d'en
- FAUX: il décide dans partir.           → d'en
- FAUX: il refuse dans discuter.         → d'en

## dans correct (lieu / temps)

- OK: il est dans la maison.
- OK: dans un bois sombre.
- OK: dans le but de partir.
- OK: rentrer dans deux heures.
- OK: il a du mal dans ce travail.

## d'en correct

- OK: il essaie d'en sortir.
- OK: il se garde d'en parler.
- OK: il est difficile d'en parler.

## Limites assumées (gaps)

### « dans le but de + infinitif »
> « dans le but de partir » est correct car l'infinitif est introduit par
> « de », pas directement après « dans ». La règle ne déclenche pas (« partir »
> n'est pas immédiatement après « dans »).
