# Corpus — confusion « dont » / « donc »

Source : reconstitué d'après Projet Voltaire, Banque de dépannage
linguistique (BDL), Question Orthographe. Aucun cas copié directement.

---

## Sens 1 — `donc` → `dont` (relative)

Pattern : NOM (sans verbe antérieur dans la phrase) + `donc` (sans virgule) +
PRONOM_SUJET + VERBE → proposition relative → corriger en `dont`.

| Fautif | Correct | Note |
|---|---|---|
| le film **donc** j'ai parlé | le film **dont** j'ai parlé | dont = pronom relatif |
| la raison **donc** elle est venue | la raison **dont** elle est venue | |
| le sujet **donc** nous discutons | le sujet **dont** nous discutons | pronom nous |
| le problème **donc** il faut parler | le problème **dont** il faut parler | |
| le livre **donc** tu m'as offert | le livre **dont** tu m'as offert | |
| un auteur **donc** on admire l'œuvre | un auteur **dont** on admire l'œuvre | |

---

## Sens 2 — `dont` → `donc` (consécutif)

Gap structurel : `, dont SUJET + VERBE` peut être une relative valide
(« l'auteur dont il parle ») ou une conséquence (rare : « il était malade,
dont il est parti »). Signal non séparable — non traité.

---

## Cas corrects — pas de suggestion

| Correct | Raison du silence |
|---|---|
| le film **dont** j'ai parlé | déjà correct |
| je suis fatigué, **donc** je dors | virgule avant donc → consécutif |
| il a fini le travail **donc** il part | verbe `fini` avant `travail` → consécutif |
| elle a terminé son projet **donc** elle se repose | idem |
| il a vu le film **donc** il part | verbe `vu` avant `film` → consécutif |
| le film **donc** lui plaisait | `lui` = pronom objet, pas sujet |
| **Donc** il faut agir | donc en tête de phrase sans nom avant |
